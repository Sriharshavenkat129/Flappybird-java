# Flappy Bird in Java

A simple Flappy Bird clone built using Java Swing and Object-Oriented Programming concepts.  
This project demonstrates game loops, collision detection, and event handling — all using basic shapes and colors (no external images).

## 🎮 Features
- Classic Flappy Bird gameplay
- Circle-based bird and colored bars as obstacles
- Score tracking
- Restart option

## 🛠️ Tech Stack
- Java
- Swing/AWT for graphics
- OOP principles
